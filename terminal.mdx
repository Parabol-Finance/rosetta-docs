## Terminal

The Terminal is an aggregated yield dashboard that displays lending and vault opportunities across multiple DeFi protocols, beyond what the Yield Router directly supports.

### Dashboard

The Dashboard tab shows HLP vault listings with current APRs, giving an overview of yield-generating opportunities across the Hyperliquid ecosystem.

### Yields

The Yields tab aggregates yield opportunities across protocols and categories:

| Category | Description |
| --- | --- |
| Lending | Supply rates from lending protocols |
| Vaults | Managed vault yields |
| Options | Options-based yield strategies |
| HLP | Hyperliquid LP vault returns |

### Supported Protocols

The Terminal pulls data from:

- Aave
- Compound
- Morpho
- Spark
- HyperLend
- HypurrFi
- Rysk

### Filtering

Use TVL filtering to focus on opportunities above minimum liquidity thresholds. This helps filter out low-liquidity positions that may not support meaningful deposits.

The Terminal is accessible from the main navigation at [app.rosetta.sh](https://app.rosetta.sh/).
