## What is Circular

Circular is Rosetta's risk intelligence layer. It provides real-time visibility into lending market risks across the protocols Rosetta routes to, transforming on-chain data into actionable risk intelligence.

Circular is available as:

- **MCP Server**: 26 tools accessible via Claude Desktop and Claude Code
- **REST API**: HTTP endpoints for programmatic access
- **Dashboard**: Visual interface at [circular dashboard](https://cairo-gules.vercel.app)

## Tools

Circular exposes 26 tools organized into three categories.

### Core Data (10 tools)

| Tool | Description |
| --- | --- |
| `get_market_overview` | Lending market overview with TVL, APY, utilization, and strategy classification |
| `get_market_detail` | Deep dive into a single market with full metrics, top positions, and risk assessment |
| `get_vault_overview` | List vaults with TVL, APY, curator info, and allocation summaries |
| `get_vault_detail` | Full vault analysis including allocations, top depositors, and concentration metrics |
| `scan_wallet_positions` | Scan all DeFi positions for a wallet across all chains and protocols |
| `get_risk_dashboard` | Protocol-wide risk overview with utilization distribution and liquidation velocity |
| `get_liquidation_feed` | Recent liquidation events with borrower details and market context |
| `get_protocol_stats` | Protocol statistics including TVL, total supply/borrow, and market count |
| `search_markets` | Search markets by token name, symbol, or strategy type |
| `get_health_status` | API health check with circuit breaker states and cache status |

## Risk Scoring

Circular uses a deterministic 0–100 risk scoring model:

| Factor | Weight |
| --- | --- |
| Protocol maturity | 15 pts |
| TVL depth | 5–50 pts |
| Utilization | 5–40 pts |
| Concentration | 5–30 pts |
| Collateral type | 5–30 pts |
| Chain risk | 5–20 pts |

Utilization above 85% is flagged as elevated risk. Above 93% is flagged as critical (withdrawal risk).

## Using Circular with Claude

Add Circular as an MCP server in Claude Desktop or Claude Code:

```json
{
  "mcpServers": {
    "circular": {
      "command": "node",
      "args": ["./packages/mcp/dist/index.js"],
      "env": {
        "CIRCULAR_API_URL": "https://cairo-gules.vercel.app"
      }
    }
  }
}
```

Once connected, you can query lending markets, scan wallet risk, simulate scenarios, and generate risk reports through natural language.
